# BioHubVenture - Programa de Mentoring 2026

Landing page del Programa de Formacion de Mentores Senior para startups de biotecnologia amazonica.

**Stack:** React 19 + TypeScript + Vite + Tailwind CSS 3 + Lucide Icons

## Run Locally

**Prerequisites:** Node.js (v18+)

1. Clone the repository:
   ```bash
   git clone https://github.com/scientificbroker/Mentoring2026.git
   cd Mentoring2026
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the app:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm run preview
```
